# *Music Theory* by Steve Stine — Module Map

The book is *Music Theory Masterclass* by Steve Stine (GuitarZoom Academy), 245 pages. the student owns the PDF separately — it isn't in this repo. This file is a stable map of what's in each module so the coach can pull one applied concept into each session.

When the coach references "Stine Module 1" or "Stine p. 12", it's pointing into this book.

## Module 1 — Notes & the Fretboard

The foundational module: the 12 notes of music, how they live on a guitar.

- **Understanding notes** — the 7 prime notes (A B C D E F G), the alphabet wraps, "circle" of notes
- **Sharps & flats (accidentals)** — # raises, ♭ lowers; mnemonic: "all notes get sharps except B and E" (spells **BE**)
- **Enharmonics** — same pitch, two names (C#=D♭, D#=E♭, F#=G♭, G#=A♭, A#=B♭)
- **Chromatic scale** — the 12 notes ascending and descending
- **Open string names** — string 6→1: **E A D G B E**
- **Pitches & direction** — toward the body = higher pitch; toward the headstock = lower
- **Half-step / whole-step** — 1 fret = half-step, 2 frets = whole-step
- **Notes on each string** — chart for strings 6, 5, 4, 3, 2, 1
- **Prime notes only** (no sharps) — useful learning shortcut
- **Cross-referencing notes across strings** — find the same note on multiple strings
- **Visual short-cut for cross-referencing** — the diagonal pattern

**Why it matters for practice:** lets the student name what he's playing. The moment he can say "this chord has the notes A C# E", the fretboard stops being abstract shapes.

## Module 2 — Music Notation, TAB, Chord Charts

How music is written down — for guitar, three notations are used in parallel.

- **Clefs** — Treble, Bass, Alto, Tenor (guitar uses Treble)
- **The staff** — 5 lines, 4 spaces; lines = EGBDF ("Every Good Boy Does Fine"), spaces = FACE
- **Middle C** — on guitar at string 2 fret 1 (also string 3 fret 5, string 4 fret 9, etc.)
- **Key signatures** (intro) — sharps/flats at the start of the staff
- **Note values** — whole (4 beats), half (2), quarter (1), eighth (½), sixteenth (¼); same chart for rests
- **Time signatures** — top = beats per measure, bottom = beat value (4/4, 3/4, 2/4, 6/8, 12/8)
- **Tempo & BPM** — Largo 40–66, Adagio 66–72, Andante 72–108, Moderato 108–120, Allegro 120–156, Vivace 156–176, Presto 176–200
- **Staccato vs. legato** — short detached vs. smooth connected
- **Alternate picking** (down ⊓ / up V) — required for fast/even playing
- **Guitar TAB** — 6 lines (1 per string), numbers = frets; 0 = open
- **Chords on TAB** — stacked numbers
- **Chord charts** — vertical fretboard view; numbers = fingers (1=index, 2=middle, 3=ring, 4=pinky); X=mute, O=open

**Why it matters for practice:** the student needs TAB and chord charts from day one. Standard notation can come later — but understanding note values and 4/4 is essential for strumming patterns.

## Module 3 — The Major Scale & Key Signatures

The single most important scale in Western music.

- **Diatonic major scale** — 7 notes + octave; Do-Re-Mi-Fa-Sol-La-Ti-Do
- **C major** — the only key with no sharps or flats (C D E F G A B C)
- **The W-W-H-W-W-W-H pattern** — every major scale follows this; deviate and you must add sharps/flats
- **Building major scales in every key** — G has 1 sharp, D has 2, A has 3, etc.
- **All key signatures**
  - Sharps (clockwise from C): G(1), D(2), A(3), E(4), B(5), F#(6), C#(7)
  - Flats (counter-clockwise from C): F(1), B♭(2), E♭(3), A♭(4), D♭(5), G♭(6), C♭(7)
  - Mnemonic: **B-E-A-D** — sharp keys spell it backwards, flat keys spell it forwards
- **Circle of 5ths** — visual map of all 12 keys; each step clockwise = up a 5th
- **Relative minor** — every major has a minor with the same key signature (C major ↔ A minor)
- **Middle C transposition for guitar** — guitar is written 1 octave higher than it sounds
- **Major scale on guitar** — closed position pattern, root on string 6 (movable); same shape plays any major scale

**Why it matters for practice:** unlocks "what chords go together in a song", lets the student transpose, and is the prerequisite for Grade 2 Module 11 (chords in a key) and Grade 3 Module 17 (major scale).

## Module 4+ — Beyond (referenced when relevant)

The book continues past Module 3 for another ~150 pages. Likely coverage (the coach should confirm by checking the PDF if the student asks):

- **Intervals** — distance between two notes (2nd, 3rd, 4th, 5th, 6th, 7th, octave), major/minor/perfect/augmented/diminished
- **Triads and chord construction** — major chord = 1-3-5, minor = 1-♭3-5, diminished = 1-♭3-♭5
- **Diatonic chords in a key** — I ii iii IV V vi vii° (uppercase = major, lowercase = minor)
- **Chord progressions** — I-IV-V, I-V-vi-IV, ii-V-I, 12-bar blues
- **Seventh chords** — major 7, minor 7, dominant 7
- **Minor scales** — natural, harmonic, melodic
- **Pentatonic scales** — major and minor
- **Modes** — Ionian, Dorian, Phrygian, Lydian, Mixolydian, Aeolian, Locrian

These come into play in Grade 2 and Grade 3. The coach should pull them in when the practice naturally surfaces them (e.g. "chords in a key" → diatonic chords; "12-bar blues" → I-IV-V and dominant 7s).
