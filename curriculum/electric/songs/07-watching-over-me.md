## Why this song is special

- Written by vocalist Matt Barlow in memory of a friend killed in an accident — the emotion is real
- Key of **E minor** — same as Enter Sandman
- Chords Em, C, G, D — all familiar by now
- Slow tempo, lots of space between notes
- E minor pentatonic — same position as Hurt and Nothing Else Matters

## What you're building

**Technique**
- Arpeggio or slow strumming — the song works either way
- Dynamics — starts quiet and builds into the chorus
- Pause and space — a key element of this song

**Theory**
- E minor — same key as Enter Sandman
- Em → C → G → D — a classic progression heard in countless ballads

**Music history**
- Iced Earth — one of the main American traditional heavy metal acts
- Song from the album *The Dark Saga* (1996), based on the *Spawn* comic
- Proof that heavy metal can be tender

## 🎸 Guitar and tone

### Which guitar to use?

**Squier Sonic Strat HSS** — whole song on one guitar, just change the selector position:

- **Verse** → position **5** (neck pickup) — warm, rounded tone, SD-1 off
- **Riff / chorus** → position **1** (bridge HB) — dense, aggressive, SD-1 on

### Yamaha THR5 — settings

| Parameter | Setting |
|---|---|
| Amp Type | CLEAN |
| Gain | 1–2 |
| Master | 4–5 |
| Tone | 5 |
| Reverb | Medium (3–4) |

### BOSS SD-1

- **Verse** — SD-1 off, THR5 CLEAN
- **Riff / chorus** — SD-1 on, THR5 CRUNCH (Drive 6–7, Tone 5–6, Level 5–6)

> 🎸 Swap guitars if you want: verse on the Tele (neck), riff on the Strat HSS (bridge HB)

## 🎯 Focus when working on it

- Start with arpeggio — same pattern as Hurt
- Rhythm one is ~130 BPM, but the ballad verses are much slower — start there
- Listen to how Matt Barlow phrases the vocal — copy those inflections on guitar
- Pause and silence — a key element

## 🎶 Improvising over this song

> 💡 **Which pentatonic fits?**
> The song is in **E minor**. Use **E minor pentatonic** — same position as Hurt (open position, 0 fret). Three ballads under one pentatonic.
