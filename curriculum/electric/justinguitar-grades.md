# JustinGuitar Grades 1–3 — Lesson Outline

Free curriculum at [justinguitar.com/categories/grade-1](https://www.justinguitar.com/categories/grade-1) (and `/grade-2`, `/grade-3`). Sequencing matters — work modules in order, don't skip ahead. Mark progress in `progress.md`, not here.

This file is a stable map of what's in each grade. Update only if Justin restructures the course.

## Before You Begin — Guitar Basics

A short orientation module before Grade 1 proper. Covers: parts of the guitar, how to hold it, how to hold a pick, tuning (with a clip-on or app), basic care.

For an electric: also learn to plug in, set amp to clean, neck pickup, volume modest. Tone knob open. No pedals yet.

## Grade 1 — First Chords, First Strums, First Songs

### Module 1: First Steps
- Posture, fretting hand position, picking hand
- Reading chord diagrams
- **First chords:** A, D, E (open)
- **First strumming pattern:** all down-strums, 4 to a bar
- **Anchor-finger trick** for fast A↔D, D↔E changes
- **Songs:** play along with backing tracks using A, D, E

### Module 2: Rhythm & Chord Changes + Your First Riff
- One-minute chord change drills
- New chords: **Em, Am**
- Air-change technique (move fingers above strings between chords)
- First riff (typically based on open low strings — a "Smoke on the Water" style two-note riff)
- Strumming pattern: down-down-up-up-down-up (the universal pattern)

### Module 3: More Chords, More Songs
- New chords: **G, D7, G7**
- Chord perfect practice (clean every string)
- Songs blending all chords learned so far
- Start naming the strings (E A D G B E) out loud

### Module 4: Note Reading on String 1 & 2
- Notes on the first string (high E): E F G A B C
- Notes on the second string (B): B C D E F
- First simple melodies
- Introduces standard notation (lightly)

### Module 5: First Rhythm Patterns & Dynamics
- Counting "1 e & a 2 e & a"
- Old faithful strumming (D D U U D U)
- Dynamics: loud/soft
- New chord: **C** (open)

### Module 6: Putting It Together
- Mini-F or A-shape practice (preparing for Grade 2)
- Songs using A, D, E, Em, Am, G, C, D7, G7
- **Grade 1 exit check:** clean changes between any two open chords at ~60 BPM

**What "passing Grade 1" looks like:** can play 3–5 simple songs end to end with the open chords above, change chords on the beat without stopping the strum, and play a basic riff cleanly.

## Grade 2 — F Chord, 16th Strumming, Riffs

### Module 7: The F Chord
- **F chord (beginner mini-barre version)** — the big hurdle
- A "mini-barre" (index across only two strings)
- Songs with F mixed into open chords
- Pinky strength exercises

### Module 8: Cool Chords, Fast Changes
- Sus2 / sus4 chord variations (Dsus2, Dsus4, Asus2, Asus4)
- One-minute change drills with F
- New strumming variations
- Slash chords introduction (e.g. D/F#)

### Module 9: Power Chords & Palm Muting
- **Power chords (5 chords)** — root on string 6, root on string 5
- **Palm muting** — picking hand technique
- Songs: rock-based progressions
- Intro to amp gain / light overdrive for power chords

### Module 10: 16th Note Strumming
- 16th note counting "1 e & a"
- 16th note strum patterns
- Funk/percussive strumming
- Muted scratch strumming

### Module 11: Chords in a Key
- **Diatonic chords** — which chords belong together in a key
- I–IV–V and I–V–vi–IV progressions
- Songwriting basics: writing a chord progression in C, G, or D
- Recognizing common progressions in songs

### Module 12: Grade 2 Songs & Consolidation
- Repertoire push: 5+ songs solid
- Transitions between songs
- **Grade 2 exit check:** clean F chord at tempo, comfortable with power chords, 16th-note strumming feel

**What "passing Grade 2" looks like:** F chord that doesn't kill the vibe, power chords with palm muting, can strum 16ths without locking up, understands which chords belong together.

## Grade 3 — Barre Chords, Pentatonic, Blues

### Module 13: The Major Barre Chord
- **E-shape barre chord** (root on string 6) — full barre
- Sliding the shape to play all major chords
- Barre chord stamina exercises
- Songs in different keys using the same shape

### Module 14: Minor Barre Chord + A-Shape
- **Em-shape barre chord** (minor, root on string 6)
- **A-shape major barre** (root on string 5)
- Switching between E-shape and A-shape forms
- New strumming patterns

### Module 15: Minor Pentatonic Scale — Position 1
- **Minor pentatonic scale, box 1** (e.g. A minor pentatonic at fret 5)
- Lead picking technique (alternate picking)
- First lead licks
- Improvising over a 12-bar backing in A

### Module 16: 12-Bar Blues
- **12-bar blues structure** (I–IV–V)
- Shuffle rhythm
- Blues in A, E, G
- Dominant 7 chords (A7, D7, E7, B7)
- Quick-change blues variation

### Module 17: Major Scale on Guitar
- **Major scale, position 1** (root on string 6)
- Connecting major scale to chord shapes you already know
- Playing simple melodies in different keys
- Ear training basics

### Module 18: Grade 3 Consolidation
- 10+ song repertoire
- Improvise short solos over blues backing
- **Grade 3 exit check:** full barre chords cleanly, minor pentatonic improv, blues feel

**What "passing Grade 3" looks like:** can play in any key using barre chords, improvise simple leads with the minor pentatonic, sit in on a 12-bar blues jam.

## How to use this with the coach

- The skill reads `progress.md` to find the **current module** and the **current lesson within it**, then runs a session around that.
- Don't jump ahead because something looks fun. Stay one module at a time, two weeks-ish per module for a 25-min-a-day pace.
- When something feels too hard, the answer is almost always: more reps at a slower tempo, not "skip and come back".
