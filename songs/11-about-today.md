## 🎶 Improvising over this song

> 💡 **Which pentatonic fits?**
> The song is in **E minor**. Use **E minor pentatonic** — same position (open) as Hurt, Zombie, and Still Loving You. Berninger's voice literally moves along these notes — find them and follow it.

| String | Frets |
|---|---|
| E (6th) | 0 — 3 |
| A (5th) | 0 — 2 |
| D (4th) | 0 — 2 |
| G (3rd) | 0 — 2 |
| B (2nd) | 0 — 3 |
| E (1st) | 0 — 3 |

> 🎤 **Vocal phrasing:** Matt sings very slowly with long pauses. Don't fill the silence — play one note, let it ring, then the next.
