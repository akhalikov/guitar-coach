## Why this song is special

- One of the most famous Russian rock ballads
- Key of **E minor** — same as Hurt, Watching Over Me, and Enter Sandman
- Chords Em, C, G, D — all familiar by now
- A very singable melody — ideal for playing vocal lines on guitar
- Alexander Vasilyev wrote it as a meditation on hopelessness — but the music sounds more like searching for a way out

## What you're building

**Technique**
- Arpeggio or soft strumming — the song works either way
- Verse → chorus → verse dynamics
- Palm muting in the heavier sections (optional)

**Theory**
- E minor — already the third song in this key in your repertoire
- Same Em → C → G → D progression as Watching Over Me
- Russian rock is built on the same Western harmonies — you can hear that clearly here

**Music history**
- Splean = one of the main Russian rock acts of the 90s and 2000s
- Vasilyev was influenced by Radiohead, Depeche Mode, Pink Floyd — you can hear it in the atmosphere

## 🎸 Guitar and tone

### Which guitar to use?

**Squier Sonic Strat HSS** — whole song on one guitar, just change the selector position:

- **Verse** → position **5** (neck pickup) — warm, rounded tone, SD-1 off
- **Chorus** → position **1** (bridge HB) — denser, SD-1 can be on

### Yamaha THR5 — settings

**Verse (SD-1 off):**

| Parameter | Setting |
|---|---|
| Amp Type | CLEAN |
| Gain | 2 |
| Master | 4–5 |
| Tone | 4–5 |
| Reverb | Medium (3–4) |

**Chorus (SD-1 on):**

| Parameter | Setting |
|---|---|
| Amp Type | CRUNCH |
| Gain | 4–5 |
| Master | 4–5 |
| Tone | 5 |
| Reverb | Off |

**SD-1:** Drive 5–6, Tone 5, Level 5–6

## 🎯 Focus when working on it

- Start with arpeggio — same pattern as Hurt
- Verse tempo ~80–90 BPM — don't rush
- Listen to Vasilyev's vocal — very singable phrases, perfect for repeating on guitar
- Pauses and silence work the same way as in Hurt — don't fill everything with notes

## 🎶 Improvising over this song

> 💡 **Which pentatonic fits?**
> The song is in **E minor**. Use **E minor pentatonic** — same position (open) as Hurt and Watching Over Me. Three ballads, one pentatonic.

| String | Frets |
|---|---|
| E (6th) | 0 — 3 |
| A (5th) | 0 — 2 |
| D (4th) | 0 — 2 |
| G (3rd) | 0 — 2 |
| B (2nd) | 0 — 3 |
| E (1st) | 0 — 3 |

- Imitate the vocal phrasing — Vasilyev sings in very guitar-like phrases
- Leave space — Russian melancholy lives in the silence between notes
