## Why this song is perfect

- Classic surf rock — teaches **fast alternate picking** on a single string
- Introduces an **Eastern / Phrygian** color — sounds exotic and cool
- Short, repeating riff — ideal for sharpening technique
- You start sounding impressive quickly

## What you're building

**Technique**
- Fast alternate picking (the main challenge)
- Runs along a single string
- Smooth position shifts
- Pick-attack control — every note the same volume

**Theory**
- The Phrygian Dominant scale (E Phrygian Dominant)
- Chromatic passing notes
- Tension and release in melody

**Tele focus**
- Bridge pickup (position 1) — bright, cutting, perfect for surf
- Dick Dale played a Strat, but the Tele's bridge pickup gives the same precise attack
- Tone knob — wide open (10)

## 🎸 Guitar and tone

### Which guitar to use?

**Squier Affinity Tele SH** — bridge single coil (position 1)

The Tele's bright, twangy bridge pickup is the perfect choice for the surf sound. The Strat's bridge can work too, but the Tele sounds more authentic.

### Yamaha THR5 — settings

| Parameter | Setting |
|---|---|
| Amp Type | CLEAN or CRUNCH |
| Gain | 2–3 |
| Master | 5–6 |
| Bass | 4 |
| Mid | 5 |
| Treble | 8 |
| Reverb | **Medium–High (5–6)** — surf tone requires reverb! |

### BOSS SD-1

- Not needed for Miserlou — keep the sound clean and bright.
- If you want a light grind: SD-1 Drive 1–2, Level 6.

## 🎯 Focus when working on it

**Picking pattern — alternate picking on every note:**

`0 ↓ • 0 ↑ • 1 ↓ • 0 ↑ • 3 ↓ • 1 ↑`

Strict down (↓) on the way down, up (↑) on the return — no exceptions.

- Original tempo is ~149 BPM. **Stage target: 25% (~37 BPM).** Speed comes by itself after the motion is automatic.
- Start in place — get the down-up motion fluent without a metronome. Then 25–30 BPM.
- The riff lives mostly on **string E** (open + fretted positions)
- Even attack — every note the same volume
