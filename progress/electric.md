## Theory (Steve Stine) — checklist

### Module 1: Notes & Fretboard
- [ ] 7 prime notes (A B C D E F G), the alphabet loop
- [ ] Open string names (E A D G B E)
- [ ] Sharps & flats, "BE" mnemonic
- [ ] Enharmonics (C#=D♭, etc.)
- [ ] Chromatic scale (ascending, descending)
- [ ] Half-step / whole-step
- [ ] Notes on string 6 (E F F# G G# A A# B C C# D D# E)
- [ ] Notes on string 5 (A B C D E F G A — prime first, then full chromatic)
- [ ] Notes on string 4 (D E F G A B C D)
- [ ] Notes on string 3 (G A B C D E F G)
- [ ] Notes on string 2 (B C D E F G A B)
- [ ] Notes on string 1 (same as string 6, two octaves higher)
- [ ] Cross-referencing notes across strings (the diagonal pattern)

### Module 2: Music Notation
- [ ] Treble clef, the staff (EGBDF, FACE)
- [ ] Middle C on guitar (string 2 fret 1)
- [ ] Key signatures (intro — sharps/flats at start of staff)
- [ ] Note values (whole, half, quarter, eighth, sixteenth) and matching rests
- [ ] Time signatures (4/4, 3/4, 2/4, 6/8)
- [ ] Tempo terms and BPM ranges
- [ ] Staccato vs. legato
- [ ] Alternate picking notation (down ⊓ / up V)
- [ ] Reading TAB (lines, fret numbers, 0 = open)
- [ ] Reading chord charts (fingers, X, O, barre notation)

### Module 3: Major Scale & Key Signatures
- [ ] Diatonic major scale (Do Re Mi Fa Sol La Ti Do)
- [ ] C major (no sharps/flats)
- [ ] W-W-H-W-W-W-H pattern

- [ ] Building D major (2 sharps: F, C)
- [ ] Building F major (1 flat: B)
- [ ] All sharp keys (G D A E B F# C#) — BEAD backwards trick
- [ ] All flat keys (F B♭ E♭ A♭ D♭ G♭ C♭) — BEAD forwards trick
- [ ] Circle of 5ths
- [ ] Relative minor (same key signature)
- [ ] Major scale on guitar — closed position pattern, root on string 6
